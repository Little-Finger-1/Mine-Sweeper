マインスイーパー


マインスイーパーを遊ぶページです

ボタン配置
Ctrl + Nボタン...ゲームを始める
Ctrl + Zボタン...マスを開ける
Ctrl + Xボタン...マスに旗を立てる
Ctrl + Cボタン...マスの旗を外す
↑↓←→ボタン...マスの移動
その他のキーには対応していないので、各自マウスで操作して下さい

※注意！！
開いたら画像のように「表示」タブから「数式バー」のチェックを外してください！マスの数字が見えてしまいます！
（さらに見出しも消すと、ゲーム画面が広くなってプレイしやすいかも...？）
